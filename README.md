<h1>python-train-ticket-check-script</h1>
python实现火车票查询  

<h1>python依赖库</h1>
```
request  
prettytable  
docopt  
pprint  
```

```
$sudo pip3 install requests prettytable docopt pprint
```

<h1>使用</h1>
```
python tickets.py 出发城市拼音 到达城市拼音 时间 1>out.txt 2>/dev/null
```
<h1>示例</h1>
```
python tickets.py jinan beijing 2016-10-15 2>/dev/null
```

就可以看到列车信息了~
